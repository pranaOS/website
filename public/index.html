Prana OS                                                                                                                                            A Unix Operating System Built Using C++ Target X86

 

PranaOS Features:

•X86 Kernel •EXT2 Filesystem •PS2 Keyboard and Mouse •UI Function by Libui

•Runtime Support For libc, libcpp, libobjc, libjs •Simple Terminal With Some Apps

System Requirements:

•i686-elf-gcc  i686 compiler to compile PranaOS •make •nasm To build asm file

 Building The Project:

•make pranaOS Builds iso •make clean removes iso/built files •make qemu To run
